# Завдання для ООП (Об'єктно-орієнтоване програмування)

## Загальна структура та вимоги завдання
- завдання виконується у окремому Jupyter Notebook
- створюється комірка Markdown з описом завдання
- створюється комірка Markdown з АІ промптом
    * завдання зроблене самостійно оцінюється вище
    * промпт для АІ має бути коректно сформульованим і давати повторюваний результат (АІ завжди робить помилку в програмі)
- створюєте кодову комірку з реалізацією індивідуального завдання
    * використовуєту AI Chat, AI Agent або самостійно
- у кодовій комірці створити клас відповідно до завдання
- реалізувати метод для обробки даних
- створити property для виведення результату
- створити метод __str__ який буде виводити Ім'я та Групу студента що виконує завдання
- у циклі створити мінімум 2-3 об'єкти класу з різними вхідними даними
- запустити кодову комірку та продемонструвати роботу класу, вивести результат
- якщо код працює коректно, завдання вважається виконаним

### Завдання 1
Створити клас `PositiveNumbersSorter`, який:
- Приймає в конструкторі список чисел (наприклад, `[1, -5, 2, 0, 6, 8, 3]`)
