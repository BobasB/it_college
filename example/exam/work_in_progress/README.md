# User Prompt
У прикріпленому файлі {PROGRAM} знаходиться навчальна програма з предмету {ACADEMIC_SUBJECT}. Там подано опис курсу, теми лекцій, лабораторних робіт та самостійної роботи. Проаналізуй цей файл, щоб зрозуміти зміст курсу та теми які викладаються.

Твоя задача створити екзаменаційні завдання для перевірки знань студентів. Завдання складаються із трьох запитань: двох теоретичних запитань та одного практичного. 

## Вимоги до екзаменаційних білетів:
- Кожен білет має містити 2 теоретичних запитання та 1 практичне завдання.
- Теоретичні запитання мають охоплювати різні аспекти тестування програмних систем та комплексів, включаючи методології тестування, інструменти, процеси та найкращі практики.
- Практичне завдання має бути спрямоване на написання коду для тестування програмних систем або комплексів з використанням Python.
- Зберігай однаковий рівень складності для всіх білетів. 
- Кожне запитання у білетах має бути унікальних, але тематика може повторюватись. Наприклад, декілька білетів можуть містити запитання про SetUp метод, але самі запитання мають бути різними, направленими на різні аспекти використання цієї констуркції у тестуванні. 
- Запитання мають мають бути оформлені простим текстом.
- Вставлені блоки коду оформлені у форматуванні Markdown.

## Вимоги до практичного завдання:
Практичне завдання складається з Python коду написаного у двох файлах: main.py та test.py.
- у першому файлі main.py потрібно буде створити функцію, у кожного індивідуальне завдання.
- у другому файлі test.py потрібно написати Unit тест для перевірки роботи функції.
- запустити програму у main.py та запустити тестування.

### Приклад запуску практичного завдання
- запускаємо програму:
    ```bash
    python main.py
    ```
- запускаємо тести:
    ```bash
    python -m unittest -v
    ```
